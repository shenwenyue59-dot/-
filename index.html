竞品关键动态总结

按品类拆分，不把面霜竞品和防晒竞品混看。

儿童面霜：夏季轻薄与晒后保湿成为主线

一页、安唯伊、松达都在把“夏天也要保湿”讲成具体场景：户外玩后脸颊干热、空调房干、孩子不愿意涂厚霜。贝乐欣则更偏 B5、敏感肌、屏障修护的专业信任感。

面霜竞品：一页 / 贝乐欣 / 安唯伊 / 松达

对我们的启发：儿童面霜内容不要只讲保湿成分，要拍“回家后薄涂”“孩子愿意伸脸”“清爽不黏”的使用证据。

儿童防晒：安全感、肤感和补涂便利是竞争焦点

戴可思、薇诺娜宝贝更强调敏感肌和专业背书；润本、贝德美偏性价比和日常通勤；安唯伊靠防晒棒/冰冰棒等形态创新；袋鼠妈妈适合亲子出游和母婴人群延展。

防晒竞品：戴可思 / 一页 / 润本 / 袋鼠妈妈 / 薇诺娜宝贝 / 安唯伊 / 贝德美

对我们的启发：儿童防晒内容要回答四个问题：会不会熏眼、会不会泛白、流汗后怎么补、回家怎么洗。

跨品类品牌：一页、安唯伊正在做组合心智

一页和安唯伊同时出现在面霜与防晒竞品池里，适合观察它们如何把“出门防晒 + 回家保湿/舒缓”做成完整护理链路。这个链路比单品卖点更容易支撑暑期内容。

组合策略

对我们的启发：可以设计“出门前防晒、回家后保湿”的双步骤内容，让儿童防晒和儿童面霜互相带动，而不是各讲各的。

平台差异：小红书做决策，抖音做场景

小红书更适合承载成分、测评、避坑、清单和真实体验；抖音更适合用亲子日常、出门流程、补涂动作和孩子反馈展示产品。两端内容不要完全同稿。
